# 인수인계

## 1. 현재 상태

- 버전: 데이터 스키마 `DATA_VERSION = 4`
- 빌드/배포: 정적 웹앱, 변경사항 커밋·푸시 예정
- 실행 URL: https://stonesilver0417.github.io/kindergarten-notebook/
- 점검: `node scripts/check-basic.cjs` 통과

## 2. 최근 작업

- 상담 질문을 최근 사례·처리 절차·담당자·부모 통보 시점을 확인하도록 구체화함.
- 교사 상호작용, 갈등 중재, 사고 통보, 생활지도, 담임 부재, 방과후·방학 돌봄을 평가 항목에 추가함.
- 후기 유형을 `사전확인`으로 명확히 하고 안전교육·점검·보험은 공시자료 확인 항목으로 변경함.
- 미입력 항목만 제외한 평균으로 영역 만점이 나오던 계산을 전체 항목 기준 비례 점수로 수정함.

## 3. 알려진 이슈

- 기존 저장 데이터는 항목 인덱스를 유지해 호환되지만, 새 항목은 미입력 상태이므로 기존 총점이 이전보다 낮아질 수 있음.
- PowerShell 실행 정책으로 `npm run check`가 차단될 수 있어 `node scripts/check-basic.cjs`로 동일 점검 가능.
