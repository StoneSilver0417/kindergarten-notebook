# 📖 소개

유치원 상담을 다녀온 후

- 상담 내용
- 관찰 내용
- 후기
- 점수
- 메모

를 하나의 앱에서 기록하고 비교할 수 있도록 만든 웹앱입니다.

별도의 회원가입 없이 브라우저에서 바로 사용할 수 있습니다.

---

# ✨ 주요 기능

## 📒 유치원 관리

- 여러 유치원 등록
- 자동 저장(LocalStorage)
- 수정
- 삭제

---

## ⭐ 평가 기능

- 5점 척도 평가
- 자동 점수 계산
- 영역별 점수
- 총점 계산

---

## 🧸 질문 리스트

상담 시 사용할 질문을 제공합니다.

- 교육철학
- 선생님
- 놀이환경
- 안전
- 현실조건

---

## 🚫 결격 체크

기본적으로 반드시 확인해야 하는 항목을 제공합니다.

- 안전
- 청결
- CCTV
- 차량
- 부모 소통

---

## 📷 메모

- 상담 내용
- 후기
- 특이사항

자유롭게 기록 가능합니다.

---

## ⭐ 비교 기능

등록한 유치원을

한 화면에서 비교합니다.

- 총점
- 영역별 점수
- 첫인상
- 결격 여부

---

## 💾 데이터 관리

- JSON Export
- JSON Import
- 자동 저장(LocalStorage)
