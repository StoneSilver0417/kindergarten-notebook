# 변경 기록

## 2026-07-15

### 변경

- 낮잠 전제 항목을 피곤하거나 정서적으로 힘든 아이의 조용한 휴식 지원 항목으로 교체.
- 관찰 평가 항목에 `관찰 못함` 선택지를 추가하고 저장·복원 지원.
- `관찰 못함`은 유치원의 결함으로 감점하지 않도록 중립값 3점으로 계산하되 평가 진행도에 별도 표시.
- 데이터 버전을 5로 상향하고 관련 자동 점검 추가.

### 검증

- `node scripts/check-basic.cjs` 통과.
- `git diff --check` 통과.

## 2026-07-14

### 변경

- 원장 교육철학과 활동 거부 질문을 실제 사례 중심으로 구체화.
- 선생님 영역에 정서 안정, 갈등 중재, 부모 소통 항목 추가.
- 안전·적응 영역에 존중 기반 생활지도, 사고 통보, 담임 부재 대응 항목 추가.
- 현실 조건 영역에 실제 돌봄 시간과 방학 중 돌봄·급식 항목 추가.
- CCTV와 시설 안전점검 결격 항목을 분리하고 확인 방법 구체화.
- `후기` 확인 유형을 `사전확인`으로 변경.
- 미입력 항목이 영역 만점과 총점 만점으로 과대 반영되지 않도록 점수 계산 수정.
- 데이터 버전을 4로 상향하고 핵심 평가 규칙 자동 점검 추가.

### 검증

- `node scripts/check-basic.cjs` 통과.
- `git diff --check` 통과.
